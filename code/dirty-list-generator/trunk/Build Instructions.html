Dirty List Generator: Build Instructions

This guide presumes that you are building on Windows. If not, presumably you are experienced enough to know what you're doing. If after following this guide, you are still having trouble compiling the Dirty List Generator, feel free to PM WrinklyNinja for help.

This guide also assumes that you are building the 'trunk' of the dirty-list-generator code. If you are building a branch, change all references to trunk\ to the directory that contains the DirtyListGenerator.sln file of the code you are building.


Preparing Your Environment/Getting Prerequisites

Download and install:

  • Microsoft Visual C++ 2010 Express Edition
  • BOOST C++ libraries - BOSS requires version 1.46.1, it will not work with earlier versions, later versions can be tried, but might also break something.
  • An SVN client of your choice.

Preparing BOSS and BOOST

  1. Perform a SVN Checkout from https://better-oblivion-sorting-software.googlecode.com/svn to a local directory of your own choice to create your working copy. (This directory shall be referred to as $BOSS from here on in.)
  2. Extract the contents of the boost-x.xx.x (x is a number) folder in the BOOST libaries archive you downloaded to $BOSS\lib\boost\ (you should have lib, include, boostrap.bat, etc. in this folder).
  3. Start a Command Prompt and cd to $BOSS\lib\boost and run bootstrap.bat
  4. In the Command Prompt, run bjam --toolset=msvc-10.0 --build-type=complete . This will take a while. Once that is done, you may close the Command Prompt window.

Building the Dirty List Generator

  1. Open $BOSS\code\dirty-list-generator\trunk\DirtyListGenerator.sln in Visual C++ 2010.
  2. Set the build type to "Release" and run Build on the DirtyListGenerator project, which will give you dirtylistgenerator.exe. Alternatively, run Debug->Build Solution to do the same.
  3. The generated exe(s) will be located in $BOSS\code\dirty-list-generator\trunk\bin\Release\.
